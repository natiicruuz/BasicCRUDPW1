# Basic CRUD system

How to run the project

1. Install dependencies
    "cors": "^2.8.5",
    "express": "^4.21.0",
    "fs": "^0.0.1-security",
    "https": "^1.0.0",
    "mongoose": "^8.6.3",
    "nodemon": "^3.1.7",
    "validator": "^13.12.0"

2. Have your own SSL certificate and add the `.key` and `.crt` files to the `cert` folder

3. npm run dev

4. enjoy!
